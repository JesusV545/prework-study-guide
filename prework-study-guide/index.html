Git

  • cd: changes directory.
  • code .: opens VS code.
  • git checkout main: switches to main branch
  • git pull origin main: "pull" command to receive a branch's modifications into the local environment. "origin" keyword indicates the source of the pull will be in the GitHub repo. "main" indicates the branch.
  • git push: pushes changes we have made locally to our remote GitHub branch.
  • git add -A: command adds modifications in the current working branch to the staging area. The "-A" flag indicates that we want to add all changes.
  • git commit -m "commit name":creates commit and captures a snapshot of the currently staged changes.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the foundation/structure of a website.
  • It is the first step to making a website.

CSS

  • In order for CSS style to be incorporated into the HTML file, it needs to be linked first.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • .card: This will create a profile and give specific characteristics to how a section will look like.
    • style.css: must be in the same folder/assets as your HTML code
  • Inline CSS: Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Internal style sheet: The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code.
  • External style sheet: This is preferred in most cases because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier

Git

  • cd: changes directory.
  • code .: opens VS code.
  • git checkout main: switches to main branch
  • git pull origin main: "pull" command to receive a branch's modifications into the local environment. "origin" keyword indicates the source of the pull will be in the GitHub repo. "main" indicates the branch.
  • git push: pushes changes we have made locally to our remote GitHub branch.
  • git add -A: command adds modifications in the current working branch to the staging area. The "-A" flag indicates that we want to add all changes.
  • git commit -m "commit name":creates commit and captures a snapshot of the currently staged changes.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • A function is a reusable block of code that performs a certian task when called.
    • A for loop will perform a task until certain criteria is met. Could even run infinitely.
    • An if statement will run a code through certains prerequisites until it meets certain criteria.
    • The command "console.log('your text here')" displays text.
    • "Var" is the key word to make a variable.
    • Just like CSS, Java must be linked to the HTML code in order to work.
